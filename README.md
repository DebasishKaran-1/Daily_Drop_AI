# DailyDrop - AI-Powered News Digest

A premium, modern web application that delivers AI-curated daily news summaries designed for busy professionals who want to stay informed in under 5 minutes.

## 🎨 Design Philosophy

DailyDrop features a **dark-mode-first** design with Apple/Stripe-level polish, offering:

- **Minimal & Elegant**: Clean layouts with generous spacing and bold typography
- **High-End Aesthetics**: Glassmorphism effects, vibrant gradients, and smooth micro-animations
- **Premium Feel**: State-of-the-art UI that feels trustworthy and calm
- **Responsive Design**: Optimized for all screen sizes

## ✨ Features

### Landing Page
- Eye-catching hero section with animated gradient text
- Interactive floating news card previews
- Feature showcase with glassmorphic cards
- "How It Works" section with animated steps
- Smooth scroll animations and parallax effects

### Daily Digest Page
- **Date Navigation**: Browse news from different days
- **Category Filters**: Filter stories by Technology, Business, Science, Politics, Health, Climate, and Sports
- **News Cards**: 
  - Category badges with unique color schemes
  - Importance indicators (high/medium/low)
  - AI-generated summaries
  - Source and timestamp information
- **Article Modal**: Full article view with enhanced readability
- **Skeleton Loaders**: Smooth loading states

### UI Components
- **Theme Toggle**: Seamless dark/light mode switching with smooth transitions
- **Sticky Navigation**: Glass-morphic navbar with blur effect
- **Category Chips**: Interactive filter buttons with hover effects
- **Importance Indicators**: Visual cues showing story significance
- **Modal System**: Premium overlay for detailed article views

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Installation

1. **Download the files** to your computer
2. **Open** `index.html` in your web browser
3. That's it! The app runs entirely in the browser

### File Structure

```
DailyDrop/
├── index.html          # Landing page
├── digest.html         # Daily digest page
├── style.css           # Complete design system and styles
├── script.js           # Interactive functionality
└── README.md           # This file
```

## 🎯 Usage

### Landing Page
- Click **"View Today's Digest"** to see today's news
- Explore features and learn how the platform works
- Toggle between dark and light themes using the moon/sun icon

### Digest Page
- **Browse Stories**: Scroll through curated news cards
- **Filter by Category**: Click category chips to filter stories
- **Change Date**: Use arrow buttons to view different days
- **Read Full Articles**: Click any news card to open the modal view
- **Toggle Theme**: Switch between dark and light modes

## 🎨 Design System

### Colors
- **Primary Gradient**: Indigo (#6366F1) to Purple (#8B5CF6)
- **Category Colors**: Unique color schemes for each news category
- **Dark Theme**: Deep blues and blacks with high contrast text
- **Light Theme**: Clean whites and grays with excellent readability

### Typography
- **Font**: Inter (Google Fonts)
- **Heading Scale**: From 0.75rem to 4.5rem
- **Font Weights**: 300-900 for various hierarchy levels

### Animations
- Fade-in effects on page load
- Smooth hover transitions on interactive elements
- Glassmorphic blur effects
- Floating card animations
- Ripple effects on button clicks
- Parallax scrolling on landing page

## 🛠️ Technical Details

### Technologies Used
- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern features including CSS Grid, Flexbox, custom properties, and animations
- **Vanilla JavaScript**: No frameworks - pure, performant JS
- **Google Fonts**: Inter font family

### Browser Compatibility
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- All modern mobile browsers

### Performance Features
- CSS-only animations (GPU accelerated)
- Intersection Observer for scroll animations
- Lazy loading support for images
- Optimized transitions with `will-change`
- Minimal JavaScript for fast load times

## 📱 Responsive Design
